无题

2020.12.31·

期末死了。

死的很彻底。

就挺离谱的。

之前觉得自己虽然浪，但不至于炸成这样。

不过既然发生了就只能接受这样的结果。

原来一直觉得退学啥的是说笑罢了，现在感觉居然有可能是真的，居然有可能发生在自己身上。

不过目前看来应该还能苟延残喘下去。

考完微积分那天回到寝室就听对门的一个老哥大喊了一声“亡羊补牢，为时未晚”。挺应景的。

往者不可谏，来者犹可追。

还是得向前看。

正好要过年了。也算一个不错的分界线。

心态好有时候也是个麻烦。

别人觉得很绝望的伤痛我一会就忘掉了。总需要什么东西一直刺激提醒自己。

虽然flag这玩意没啥用，但是立还是得立的。总得设立一些个目标，不管能不能完成都至少有个方向。

这场期末确实值得好好反思一下的。等到全考完了得写一段啥总结一下。

终于体会到不敢面对父母老师同学的感觉了。

还是得面对。

真正丢脸的不是被别人发现爆炸的事实，而是逐渐走向爆炸的过程。

过程都做了又怕个啥呢。改正就好了。毕竟能力感觉还是跟的上的，不是硬件问题。

2020-12-14

2020年度总结

2020年度总结·

好像是第一次写这种东西。

主要原因是大一上感觉有很多东西没有思考清楚。想开一个东西让我多思考思考。不过看到大家都写这种年度总结于是就跟风写一段好了。

首先2020年上半年。我不是很想写这一段。反正就经过了一个挺有趣的高考准备以及高考，莫名其妙考得很好，以为自己多么多么牛逼，到了大学才发现还有更牛逼的。

所以永远不要自以为是。

暑假度过的挺混蛋的。感觉没学到啥东西。或者说学了不少东西但是效率还是有些低。

好了我们上半年就这样过去。（为啥不直接写一个大一上总结呢）

其实想要写总结的起因是微积分考的太差了。

考完的时候我真的是崩溃的。然后我不好意思表现出来。

于是回寝之后整个下午加晚上的前一半就没讲话。

其实很简单。就是准备的时间太短了。

考前一直以线代为主，考试的上午才开始看微分方程。我当时还自以为没啥问题。

真·自闭

不过也算一个好事。至少后来的考试都没再这么自闭（

感觉失败的原因：

  1. 平时没有跟着做习题课的题，考前才做，事实上这些题质量确实挺高，值得做的
  2. 对自己的实力做了错误估计
  3. 练习太少。这是主要原因。微积分这种课程需要大量的习题来提升熟练度。期中的顺利给了我错误的印象

下学期的努力改正方向：

  1. 平时ddl尽量下发当天完成，甚至可以提前完成
  2. 平时多做练习，尤其微积分、抽代、智应数这种立刻学科感觉不做题要凉
  3. 合理规划时间，不要再像这次一样考前疯狂消耗生命搁这复习
  4. 多去图书馆吧（

然后日常的一些思考：

首先下学期不再要考3km了，感觉可能因此跑步水平大幅下降

其次要引体，这个能不能坚持训练实在是个问题

上课效率太低

得坐前排认真听课

不翘了（

最后就是多看看课外书啥的

学习与养生并重（

我总觉得得开发新的娱乐方式 我预料到光靠看课外书撑不住这个2021

最后就加油吧😺
