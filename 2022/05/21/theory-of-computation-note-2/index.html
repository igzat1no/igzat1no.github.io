Time complexity·

Definition·

  • Let M be a deterministic Turing machine that halts on all input. The running time or time complexity of MMM is the function f:N→Nf: N\rightarrow Nf:N→N, where f(n)f(n)f(n) is the maximum number of steps that MMM uses on any input of length nnn.

Time complexity class·

  • TIME(t(n)): languages decidable by O(t(n))O(t(n))O(t(n)) time deterministic TM
  • NTIME(t(n)): languages decidable by O(t(n))O(t(n))O(t(n)) time nondeterministic TM
  • NP=∪kNTIME(nk)NP = \cup_k NTIME(n ^ k)NP=∪k​NTIME(nk) 的理解：
    • NP 是能在多项式时间内验证答案正确性，那么可以用 非确定性图灵机 猜一个答案 然后多项式时间内验证他 猜的时候可以保证如果有答案就能猜到答案 所以 NP ⇔\Leftrightarrow⇔ NTIME

The Cobham-Edmonds thesis·

  • For any realistic models of computation M1M_1M1​ and M2M_2M2​, M1M_1M1​ can be simulated on M2M_2M2​ with at most polynomial slowdown.
  • 只有多项式复杂度的东西才能够在实际计算机上计算

Space complexity·

Space complexity class·

  • For any function f:N→Nf:N \rightarrow Nf:N→N, we define:

    • SPACE(f(n)) = { L : L is decidable by a deterministic TM M which scans O(f(n))O(f(n))O(f(n)) tape cells}
    • NSPACE(f(n)) = { L : L is decidable by a non-deterministic TM M which scans O(f(n))O(f(n))O(f(n)) tape cells on any branch of its computation}
    • NTIME(f(n)) 就是说 模拟的步骤 时间是 f(n)
    • NSPACE(f(n)) 就是说 模拟的步骤 空间是 f(n)

  • Example:

    • SAT ∈\in∈ SPACE(n)

      can be solved in linear space

Log-space reduction·

  • AAA is log-space reducible to BBB, written A≤LBA \leq _L BA≤L​B, if there exists a log space TM MMM that, given input www, outputs f(w)f(w)f(w) s.t. w∈Aw\in Aw∈A iff f(w)∈Bf(w) \in Bf(w)∈B.
